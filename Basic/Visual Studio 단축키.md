# 디버그
Ctrl+F5      : 디버그 하지 않고 시작   
F5           : 디버깅 모드   
F9           : 중단점 설정/해제   
F10          : 프로시저 단위로 실행   
F11          : 한줄 단위로 실행   

# 빌드
Ctrl+Shift+B : 빌드

# 편리기능
F12          : 선택한 함수로 이동

Ctrl+-       : 이전 커서 위치로 이동   
Ctrl+Shift+- : 다음 커서 위치로 이동

Ctrl+M+M     : 선택영역 개요 확장/축소   
Ctrl+M+L     : 파일 전체영역 개요 확장/축소

Ctrl+K+D     : 파일 전체영역 자동들여쓰기   
Ctrl+K+F     : 선택영역 자동들여쓰기

Ctrl+K+C     : 선택 범위 주석화   
Ctrl+K+U     : 선택 범위 주석 해제
